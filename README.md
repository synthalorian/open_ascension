## ✨ What is Open Ascension?

**Open Ascension** is a fully-featured companion app for the [Project Ascension](https://ascension.gg/) custom WoW server. It lets you plan and build classless characters with **real game data** scraped from the official Ascension resources, browse mystic enchants with their unlock locations, track M+ affix rotations, explore dungeon routes, browse item affixes, and much more.

All data is **embedded in the app** — no network connection required once installed. The app is a direct companion to [ascension.help](https://ascension.help/), the Worldforged reference database.

## 🔥 Features

### Character Builder
- Select any of **9 classic WotLK classes** and **3 specs each**
- Browse **266 abilities** filtered by class
- Visual **talent tree** with connecting lines and prerequisite enforcement (195 talents across 27 specs)
- Pick mystic enchants and gear — watch your **stats update in real-time**
- **11 races** with racial bonuses integrated into stat computation
- WotLK stat formulas (Str → AP, Agi → Armor, Sta → HP, etc.)

### Mystic Enchants
Browse 176+ real mystic enchants scraped from [ascension.help](https://ascension.help/):
- **Search** by name or keyword
- **Filter** by zone/location
- Full descriptions with exact chest locations, mob drops, and unlock conditions
- Real enchants like Soulbender, Tools of War, Locust Ranger, Forbidden Technique, Dragon Warrior, and many more

### Item Affixes
Complete reference for all **56 crafting affixes** across **8 stat categories**:
| Category | Affixes |
|----------|---------|
| 💪 Strength | 9 affixes (of Strength, of Swift Might, of Critical Might, etc.) |
| 🏃 Agility | 10 affixes (of Agility, of the Monkey, of Sharp Strikes, etc.) |
| 🧠 Intellect | 6 affixes (of Intellect, of the Eagle, of Prophecy, etc.) |
| ❤️ Stamina | 8 affixes (of Stamina, of the Unyielding, of Quick Defense, etc.) |
| 🌊 Spirit | 6 affixes (of Spirit, of the Soulweaver, of Nature's Grace, etc.) |
| ⚔️ Attack Power | 4 affixes (of the Deathblow, of the Ravager, etc.) |
| 🔮 Spell Power | 8 affixes (of Spell Power, of Arcane Dominance, etc.) |
| 🛡️ Tanking | 5 affixes (of Blocking, of Dodging, of Parrying, etc.) |

### M+ Upgrade Costs
Complete **Mythic+ upgrade cost schedule** from ascension.help — 41 tiers (Purchase through Mythic 40) across 8 item slot categories with a totals row. Costs in Emblems of Triumph.

### Mythic+ Affixes
All **9 Ascension-specific affixes** with detailed mechanics:
- Tyrannical (Mythic 2+), Fortified (Mythic 2+)
- Necrotic (Mythic 4+), Killer Bees (Mythic 7+), Pack Tactics (Mythic 7+)
- Sanguine (Mythic 10+), Corpse Explosion (Mythic 10+)
- Titanic Power: Glacial (Mythic 14+), Titanic Power: Infernal (Mythic 14+)
- Full descriptions, Ascension-specific notes, and strategy tips

### Dungeon Routes
**13 TBC dungeon routes** with invis skip strategies and route notes:
- Arcatraz, Auchenai Crypts, Hellfire Ramparts, Mana Tombs
- Mechanar, Sethekk Halls, Shadow Labyrinth, Shattered Halls
- Slave Pens, Steamvaults, The Blood Furnace, The Botanica, Underbog
- Detailed strategy notes for each dungeon
- Route images (WIP)

### Gear Database
Comprehensive item database covering **200 items** from:
- **Naxxramas** (T7), **Ulduar** (T8), **Trial of the Crusader** (T9), **Icecrown Citadel** (T10)
- **Emblem of Triumph/Frost** gear
- **PvP Arena** (Season 8 Deadly Gladiator)
- **Profession-crafted** (Blacksmithing, Engineering)
- **Reputation vendors** (Kirin Tor, Wyrmrest, Argent Crusade)
- **Heroic dungeon** drops

Filter by slot, rarity, type, armor type, weapon type, or source.

### WeakAuras
Community WeakAura sharing platform (WIP — contribute on [ascension.help](https://ascension.help/)):
- Category tabs: All, M+, Raids, QoL, Builds, PvP, Other
- Search and filter
- Import string sharing

### Build Sharing
- Export builds as **base64 shareable codes** — copy to clipboard and paste anywhere
- Import single builds or batch import multiple at once
- Conflict detection with automatic rename for duplicates

### Lore Browser
**42 lore entries** across 6 categories:
- Characters, Locations, Factions, Artifacts, Events, Ascension-specific content
- Rich descriptions with themed presentation

### Realms
**3 Ascension realms** with server info:
- Area 52, Elune, BoneBread
- Server status and connection info

### Themes
**11 custom themes** to personalize your experience:
- Core synthwave aesthetic + lore-inspired themes
- Settings with data export, import, and clear

## 🚀 Quick Start

### Web App (No install)
Visit **[synthalorian.github.io/open_ascension](https://synthalorian.github.io/open_ascension/)** — works in any modern browser.

### Android
Download the latest APK from [Releases](https://github.com/synthalorian/open_ascension/releases/latest) and install.

### Linux Desktop
```bash
git clone https://github.com/synthalorian/open_ascension.git
cd open_ascension
flutter pub get
flutter build linux --release
./build/linux/x64/release/bundle/open_ascension
```

### Build from Source
Requires [Flutter](https://flutter.dev) installed.

```bash
flutter pub get
flutter build linux --release    # Desktop
flutter build web --release      # Web app
flutter build apk --release      # Android APK
```

## 🛠️ Tech Stack

- **Flutter 3.41.9** — cross-platform UI (Android, web, Linux, iOS)
- **Riverpod** — state management
- **go_router** — declarative routing with fade transitions
- **JSON serializable** — model codegen
- **SharedPreferences** — local persistence
- **flutter_animate** — micro-animations
- **shimmer** — loading skeletons
- **cached_network_image** — image caching

## 📂 Project Structure

```
lib/
├── core/                   # Theme, navigation, utilities
│   ├── navigation/         # app_router.dart — all routes
│   ├── theme/              # 11 custom themes + theme provider
│   └── utils/              # Base64 build codes
├── data/                   # Models + data
│   ├── models/             # All data models
│   │   ├── ascension/      # Affix, DungeonRoute, Weakaura, etc.
│   │   └── (class, race, ability, talent, build, gear, etc.)
│   └── repositories/       # AppDatabase + AscensionRepository (Riverpod providers)
└── features/               # Feature screens
    ├── home/               # Dashboard + app drawer
    ├── class_builder/      # Abilities, talents, enchants, gear, stats
    ├── talent_tree/        # Visual talent tree with connections
    ├── mystic_enchant/     # Enchant browser by zone/name
    ├── gear/               # Gear database with multi-filter
    ├── builds/             # Saved build manager + share/import
    ├── affixes/            # Mythic+ affixes + Item affixes browser
    ├── dungeon_routes/     # M+ dungeon route guides
    ├── weakauras/          # Community WeakAura sharing
    ├── suggestions/        # Feature suggestions & voting
    ├── items/              # Item affix suggestions
    ├── mplus/              # Mythic+ affix rotation + upgrade costs
    ├── lore/               # Lore browser + entry detail
    ├── realms/             # Realm browser
    ├── settings/           # Theme selector, data management, about
    └── static/             # Static pages (About, Contact)

assets/
├── data/                   # All JSON data files
│   ├── mystic_enchants.json    # 176 enchants
│   ├── item_affixes.json       # 56 item affixes
│   ├── mplus_upgrades.json     # 42 upgrade tiers
│   ├── mythic_affixes.json     # 9 M+ affixes
│   ├── dungeon_routes.json     # 13 dungeon routes

│   └── (enchants, suggestions, weakauras, tags, etc.)

```
